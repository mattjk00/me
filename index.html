Personal Projects

Cellm 2022, In Early Development

A programming language for designing cellular automata. Inspired by Conway's Game of Life, this software package allows developers to create their cellular simulations using a simple programming language. The parser and interpreter are being developed using Rust. The library is designed to be renderer agnostic, so developers can visualize their simulations however they want.

Mono Catcher 2021

Mono Catcher is an audio utility for catching 'fake' stereo files. It was created at the request of an Audio Engineer friend who would often spend hours converting faux-stereo stems that musician clients would send him. The software will automatically convert any stereo files with identical channels to mono. Developed using C and C++.

Vino Art Files 2021

A pure C library for simple image/audio file IO. Designed for creative coders who need an easy to use and cross platform library that can load many different types of image and audio files. The library also provides functions for programmatically editing and converting these files.

Crypto Currency Prototype 2021

This project was developed for education purposes. It is a functioning crypto currency implemented in Rust based on the bitcoin whitepaper. Peer to peer networking was implemented using libp2p. Although the project was not intended for commercial use, security and data integrity was an important focus. SHA256 encryption is used and the blockchian verifies all newly mined nodes and transactions.

Academic Projects

Smart Library Computer Vision Spring 2022

An image processing project for recognizing call labels on the spines of books in a library. My responsabilities included creating a convolution neural network for image classification and creating detailed documentation. Datasets were created using the Roboflow platform and the CNN was trained using PyTorch. My work helped to increase the accuracy of the computer vision model to over 90% accurate.

Differential Equation Visualizer Fall 2021

An education tool for Applied Math and Differential Equations. Developed under the direction of a mathematics professor for use as an education tool for upper level applied math courses. The web interface was developed using React.js and Three.js. The user may input boundary conditions for a Laplace equation. The application will parse these boundary conditions and compute the discrete Laplace equation. These results are then used to construct a 3D mesh using Three.js and are displayed on the screen.

Uses of Genetic Algorithms in Machine Learning Fall 2021

This informal research paper was created for a student seminar in machine learning. The paper explores how genetica algorithms can be used in optimization problems such as feature selection and parameter estimation. To accompany the paper, I programmed an application in Rust that solves the Traveling Salesman problem using a genetic algorithm. I developed other genetic algorithms during my research that did not get included in the final paper. These included a genetic algorithm for factoring semiprimes (this did not yield efficient results) and an algorithm for computing a nonlinear regression on a dataset.
